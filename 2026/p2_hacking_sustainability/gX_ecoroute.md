summary: "EcoRoute is a lightweight command-line tool that helps developers reduce the energy consumption of AI-assisted coding. Developers often default to large language models for generating code, even for simple tasks. Because larger models require significantly more computation, this practice leads to unnecessary energy use and increased environmental impact. EcoRoute addresses this problem by recommending the most energy-efficient AI model for a given coding prompt. Users submit a prompt through the CLI, and EcoRoute analyzes its complexity using a lightweight classifier. Based on this analysis, the tool recommends an appropriate model tier (e.g., small or large). The CLI then displays key information to the user, including the recommended model, estimated energy consumption, baseline energy if a large model were used, and estimated energy savings. To evaluate EcoRoute, we use the HumanEval benchmark for code generation tasks. We compare two scenarios: always using a large model versus routing prompts through EcoRoute. Performance is measured by code correctness (pass rate) and estimated energy consumption. The objective is to show that EcoRoute can maintain comparable accuracy while reducing the overall energy required for AI-assisted programming."
