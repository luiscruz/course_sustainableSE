What is GDPR? How can I be GDPR compliance? If these are questions that you asked yourself as a CEO/developer, use our tool to find answers!

Unfortunately, not all companies are compliant with the GDPR. The recent Odido data breach showed that, since the company stored the data of older customers longer than they should. And they are not the only ones, as many others do the same thing. While we don't know if this was done with malicious intent or out of ignorance, we propose a tool that CEOs and developers can use to learn about GDPR and check if they are compliant with it.

The tool will present information about the GDPR, showing and explaining the rules. It will also have a quiz where people can test their knowledge and companies can check if they are GDPR compliant.
